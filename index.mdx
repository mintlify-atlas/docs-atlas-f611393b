## What agentic-stack gives you

agentic-stack is not just a folder template — it ships the files that go *inside* the folder: seed skills, memory management scripts, protocol files, and adapter shims for eight harnesses.

<CardGroup cols={2}>
  <Card title="Memory system" icon="database" href="/memory/overview">
    Four layers — working, episodic, semantic, personal — each with its own retention policy and query-aware retrieval.
  </Card>
  <Card title="Dream cycle" icon="moon" href="/memory/dream-cycle">

## Get started in three steps

<Steps>
  <Step title="Install">
    ```bash
    brew tap codejunkie99/agentic-stack https://github.com/codejunkie99/agentic-stack
    brew install agentic-stack
    ```
  </Step>
  <Step title="Pick your harness">
    ```bash
    cd your-project
    agentic-stack claude-code
    # or: cursor | windsurf | opencode | openclaw | hermes | pi | standalone-python
    ```
  </Step>
  <Step title="Complete the onboarding wizard">
    Answer six questions about your workflow preferences. The wizard writes `.agent/memory/personal/PREFERENCES.md` — the first file your AI reads every session.
  </Step>
</Steps>

<Tip>
  Run `agentic-stack claude-code --yes` to skip the wizard and accept all defaults — useful in CI or when setting up multiple projects quickly.
</Tip>
